# Ferretería Golden — Sistema Web (Proyecto UNI 2025)

**Proyecto:** Sistema Web para Ferretería Golden  
**Carrera:** Ingeniería en Sistemas (UNI)  
**Integrantes:** Hanssel Jacobo Miranda Díaz, Sheyla Marleth López Pérez  
**Docente:** Eliab Javier Selva Cruz  
**Fecha:** 7 de noviembre de 2025

## Resumen
Aplicación web para la gestión de una ferretería: productos, inventario, clientes y ventas. Construida con **React + Vite** en el frontend y **Firebase (Firestore + Auth)** en el backend.

## Tecnologías
- React
- Vite
- Firebase (Firestore, Auth)
- React-Bootstrap
- Netlify (despliegue)

## Estructura de carpetas
src/
├─ components/
├─ views/
├─ database/
├─ App.jsx
└─ main.jsx

## Ejecutar en local
1. Clonar repo
2. `npm install`
3. Crear archivo `.env` con variables de Firebase (Vite usa `VITE_...`)
4. `npm run dev`

## Despliegue
- Conectar repositorio a Netlify y configurar build command `npm run build` y `dist/` como carpeta pública.
