# mi-proyecto-vite

Proyecto inicial creado con Vite + React para la asignación de Diseño de Sistemas en Internet.

## Ejecutar local
1. npm install
2. npm run dev
3. Abrir http://localhost:5173

## Build producción
npm run build
npm run preview

## Notas
- Archivo de redirect para Netlify: public/_redirects (/* /index.html 200)
- Bootstrap y Bootstrap Icons instalados e importados en src/main.jsx
